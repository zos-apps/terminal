## About

Command-line interface for system access and scripting.

## Features

- Multiple tabs
- Themes
- Shell selection
- Command history
- Auto-complete

## Installation

### From App Store
Open the **App Store** in zOS and search for "Terminal".

### Via npm
```bash
npm install @zos-apps/terminal
```

## Usage

- Type commands to execute
- Tab for auto-complete
- Use arrow keys for history

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd+T` | New tab |
| `Cmd+W` | Close tab |
| `Ctrl+C` | Cancel |


## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
