# ⬛ Terminal

> Command-line interface with zsh-like shell

Part of the [zOS Apps](https://github.com/zos-apps) ecosystem.

## Installation

```bash
npm install github:zos-apps/terminal
```

## Usage

```tsx
import ZTerminal from '@zos-apps/terminal';

function App() {
  return <ZTerminal />;
}
```
